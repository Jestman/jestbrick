# Yol haritası notları

Faz planının dışında, sohbetlerde doğan fikirler. Kaynak: Yusuf.

## Rozetler / Gamification (Faz 2-3)
Eşiklere ulaşınca otomatik rozet:
- **Parça sayısı:** 1.000 / 10.000 / 50.000 / 100.000 parça
- **Set sayısı:** 5 / 10 / 25 / 50 / 100 set
- **Minifigür sayısı:** 10 / 50 / 100 / 500 figür
- **Belirli minifigür:** nadir/ikonik figüre sahip olmak (ör. Legolas, CMF serisi tamamlama)
- Mevcut rol rozetlerinin (★ İçerik Üreticisi, ✓ Güvenilir Satıcı) yanında görünür
- Profilde vitrin + akışta "X rozeti kazandı" aktivitesi (paylaşım tetikleyici → viral döngü)
- Teknik: `badges` tanım tablosu + `user_badges` + koleksiyon değişiminde eşik kontrolü

## Diğer
- Minifigür bazlı istek listesi (set gibi figür de istenebilsin — eşleştirme motoru figürlere de çalışır)
- Set detayında "B-model MOC'lar" (Rebrickable alternates, tıklayınca yükle)
- Brickset entegrasyonu: emeklilik tarihi + fiyat (Faz 2 planında var)
